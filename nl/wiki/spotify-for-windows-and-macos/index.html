Spotify for Windows and Mac OS

    What the API means and how it works

  • By Mario Ortiz Manero at 5 April 2020

Dit is de standaard API voor Windows en Mac OS gebruikers. Deze API gebruikt de SwSpotify library om informatie op te vragen van de desktop Spotify apps op Windows en Mac OS. Je hebt geen extra libraries nodig, omdat deze automatisch worden geïnstalleerd voor deze besturingssystemen.

De app is niet getest op Mac OS, omdat de emulators hiervoor nogal gelimiteerd zijn. Mocht je problemen ondervinden, open dan een issue met gedetailleerde informatie over je probleem.

Opmerking: deze API werkt in principe ook op Linux, maar deze is uitgeschakeld op Linux omdat de Spotify for Linux API beter is in de meeste gevallen.

About

Vidify is an open source project at its core. Anyone can contribute and improve it on GitHub. It’s under active development, releasing new fixes and features frequently. Don’t forget to stay updated.
